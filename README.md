# Метод продолжения по параметру для решения краевых задач
В рамках курса «Практикум на ЭВМ» была поставлена задача изучить метод продолжения по параметру для решения краевых задач и написать программу с пользовательским интерфейсом на языке Python, реализующую данный метод. Для решения внутренней и внешней задачи используется явный метод Рунге-Кутта 3 прядка аппроксимации. Эффективность и работоспособность данного метода были проверены на нескольких известных примерах: краевая задача двух тел, предельные циклы в системе Эквейлера, функционал типа "энергия"для трёхкратного интегратора.
